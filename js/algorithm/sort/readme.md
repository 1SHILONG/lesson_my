- 如何写出一个高质量的冒泡排序
    1. 当在一次冒泡的内层循环中， 一次都没有发生
        通过检测本轮是否有交换？ 如果没有，待排序区域已经有序 就可以退出 isSorted 布尔值
    2. 基于isSorted 布尔值优化

- 插入排序的关键点
    1. 当前元素前面的那个序列是有序的。 有序区间 
    2. 所有在当前元素前面的数都不大于它， 所有在当前元素后面的数都不小于它
    3. 一个个往后移动， 让我们待排序的元素插入新的位置
